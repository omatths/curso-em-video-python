# 🐍 Desafios de Python - Curso em Vídeo

Este repositório foi criado para organizar e documentar a minha evolução durante o curso de **Python 3**, ministrado pelo professor Gustavo Guanabara no canal *Curso em Vídeo*.

Em vez de criar dezenas de arquivos separados, optei por centralizar os desafios em um formato de **Jupyter Notebook (.ipynb)**. Isso me permite manter o enunciado de cada exercício, o código em Python e o resultado da execução organizados em um único documento interativo!

## 🚀 Como Visualizar os Exercícios

Como os códigos estão salvos em um arquivo `.ipynb`, você tem duas formas de visualizar:

1. **Direto pelo GitHub:** O próprio GitHub consegue renderizar notebooks do Jupyter. Basta clicar no arquivo `exercicios.ipynb` para ver todo o código formatado e as saídas de execução.
2. **No seu VS Code local:** 
   * Certifique-se de ter a extensão **Jupyter** instalada no seu VS Code.
   * Abra a pasta do projeto e abra o arquivo `exercicios.ipynb`.
